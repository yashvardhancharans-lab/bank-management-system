# 💳 Bank Management System (Python)

A simple command-line banking system built using Python and JSON for data storage.

---

## 🚀 Features

- 🆕 Create Account
- 💰 Deposit Money
- 💸 Withdraw Money
- 📄 View Account Details
- ✏️ Update Account Information
- ❌ Delete Account

---

## 🛠️ Tech Stack

- Python
- JSON (for database)
- OOP (Object-Oriented Programming)
